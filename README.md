 Monthly Purchase Analysis using Python
 Project Overview
This project focuses on analyzing purchase transaction data and visualizing the total purchase amount on a monthly basis using Python and Jupyter Notebook. The main objective is to understand sales trends over time and present them in a simple and easy-to-understand graphical format. This project is suitable for beginners who are learning data analysis and data visualization.

 Objectives
To process and clean purchase data

To convert purchase dates into monthly format

To analyze monthly purchase trends

To calculate total purchase amounts for each month

To visualize results using line charts

To improve understanding of Pandas and Matplotlib

 Technologies and Tools Used
Python Programming Language

Jupyter Notebook

Pandas Library for Data Analysis

Matplotlib Library for Data Visualization

 Project Features
Automatic conversion of date columns into datetime format

Monthly grouping of purchase data

Accurate calculation of total purchase amounts

Interactive and clear line chart visualization

Well-structured and readable code

Beginner-friendly implementation

 Data Processing Steps
Load the dataset into a Pandas DataFrame

Convert the Purchase_Date column into datetime format

Extract month and year information from dates

Group the dataset by month

Calculate the total purchase amount for each month

Prepare data for visualization

 Visualization Details
The project uses a line chart to represent the total purchase amount per month.

X-axis represents months

Y-axis represents total purchase amount

Each point shows monthly sales

Helps in identifying increasing or decreasing trends

 How to Run the Project
Clone this repository to your local system

Open Jupyter Notebook

Navigate to the project folder

Open the notebook file

Run all cells to view the results

 Project Structure
task9.ipynb – Contains the complete analysis and visualization code

README.md – Project documentation

 Expected Output
After running the notebook, a line chart will be displayed showing the total purchase amount for each month. This visualization helps users understand monthly sales performance clearly.
